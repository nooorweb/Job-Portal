⚙️ FUNCTIONALITY REQUIREMENTS
State Management (React)
- activeJob          → currently viewed job object
- activeTab          → 'overview' | 'syllabus' | 'quiz'
- bookmarkedJobs     → Set of job IDs (localStorage)
- searchQuery        → string
- activeFilters      → { category, type, level }
- sortBy             → 'latest' | 'salary' | 'relevance'
- quizState          → { started, answers[], currentQ, timeLeft, submitted, score }
- syllabusProgress   → { [jobId]: { completedWeeks: [] } }
Search & Filter Logic

Filter JOBS array client-side by: title, tags, company, category, type, level
Debounce search input by 300ms
Sort functions: by postedDate, by salary (parse range), by relevance (match score)

Quiz Engine

Routing (React Router v6)

/jobs/:id           → Job Detail (default tab: overview)
/jobs/:id/syllabus  → Job Detail (syllabus tab)
/jobs/:id/quiz      → Job Detail (quiz tab)
